Description List Questions

Define HTML
HTML, also known as (Hyper Text Markup Language) is a front-end program that is used to make webpages and usually works in coordination with Javascript(Background Program) and CSS(Styling)

Define CSS
CSS, also known as (Cascading Style Sheets), describes how HTML elements are to be displayed on screen, paper, or in other media. It saves a lot of work in styling and usually uses an external stylesheet in CSS files.

Define Javascript
Javascript is a backend program that provides the functionality for HTML pages

What is Responsive Design
Responsive Web Design is about using HTML and CSS to automatically resize, hide, shrink, or enlarge, a website, to make it look good on all devices (desktops, tablets, and phones)

What is Accessibility
Accessibility in HTML is about using elements for their correct purpose. For intance, a div class is used to group things together, it cannot be used as a button. Similarly, a button element cannot be used as a method to group elements together. In other words, it means making each element serve its rightful purpose
